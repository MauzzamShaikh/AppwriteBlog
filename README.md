# 📝 My Blog

A modern full-stack blog application built with **React** and **Appwrite** as part of my React learning journey.

This project helped me understand how to build a real-world CRUD application using React, state management, authentication, cloud storage, and database services.

---

## 🚀 Features

- 🔐 User Authentication (Sign Up / Login / Logout)
- ✍️ Create Blog Posts
- 📝 Update Existing Posts
- ❌ Delete Posts
- 📖 View Individual Posts
- 📚 Browse All Posts
- 🖼️ Image Upload using Appwrite Storage
- 📝 Rich Text Editor using TinyMCE
- 👤 Author-only Edit/Delete Controls
- 📱 Responsive User Interface
- ⚡ Fast React + Vite Development

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/my-blog.git
```

Navigate into the project

```bash
cd my-blog
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the root directory.

```env
VITE_APPWRITE_URL=
VITE_APPWRITE_PROJECT_ID=
VITE_APPWRITE_DATABASE_ID=
VITE_APPWRITE_COLLECTION_ID=
VITE_APPWRITE_BUCKET_ID=
VITE_TINYMCE_API_KEY=
```
